## 구현 완료 기능 목록

### 화면 흐름
- [x] 신규간호사 사번·성명 입력 화면
- [x] 역할 선택 화면 (프리셉티·프리셉터·교육전담·수간호사)
- [x] 4주·8주 선택 화면
- [x] 교육자 정보 입력 화면 (사번·성명, 역할에 따라 다름)
- [x] 체크리스트 입력 화면

### 체크리스트 입력
- [x] 0~3점 버튼 입력
- [x] 첫 점수 입력 시 전체 미입력 문항 일괄 자동 적용 (B안)
- [x] 교육 실행일 전체 일괄 입력
- [x] 교육 실행일 문항별 개별 입력
- [x] 역할별 표시 문항 필터링
- [x] 교육전담 문항 시각적 구분 (노란 테두리 + 배지)
- [x] 자가평가 점수 타 역할 열람 (프리셉터·교육전담·수간호사)
- [x] 교육자 평가 점수 수간호사 열람

### 서명
- [x] 전자서명 (손가락·터치펜·마우스)
- [x] 서명자 이름 입력
- [x] 점수 입력된 문항 전체 일괄 서명
- [x] 기존 서명 재사용 (재서명 없이 재활용)
- [x] 서명 이미지 Google Drive 전자서명 폴더에 PNG 저장

### 저장 및 불러오기
- [x] LocalStorage 자동저장 (800ms debounce)
- [x] 임시저장 — Google Drive JSON + XLSX 동시 저장
- [x] 임시저장 중 팝업 ("임시저장 중...")
- [x] 임시저장 완료 팝업 ([처음으로] [계속 작업])
- [x] Drive 저장 세션 목록 불러오기
- [x] 대상자별 저장 키 분리 (사번 기준)

### 제출 흐름
- [x] 미답변 문항 제출 차단 — 미입력 문항 번호 목록 표시
- [x] 최종 제출 확인 팝업 — 총점·평가일·교육자 성명·사번 표시
- [x] 최종 제출 후 처음 화면(신규간호사 입력)으로 이동
- [x] 70점 미만 경고 팝업 (수간호사)
- [x] 역할별 독립 제출 및 잠금 (`submittedRoles`)

### 엑셀 출력
- [x] 원본 템플릿 기반 XLSX 출력 (ver.202605)
- [x] 자가평가(J열)·교육자평가(K열)·수간호사평가(L열) 점수 입력
- [x] 교육 실행일(H열)·서명자명(I열) 입력
- [x] 프리셉터·수간호사 전자서명 이미지 삽입
- [x] 집계표 자동 계산 (0~3점 분포, 합계, 총점)
- [x] 표지 시트 대상자 정보 입력 (최종제출 시만)
- [x] 체크리스트 헤더 교육기간·부서·성명·사번 입력
- [x] 통일 파일명 생성 (`{사번}_{역할}_{교육자성명}_{사번}_{상태}.{ext}`)
- [x] 임시저장 엑셀은 고정 파일명으로 덮어쓰기, 최종제출은 별도 저장
- [x] ExcelJS 청크 동적 분리 (메인 번들 291KB, ExcelJS 936KB lazy load)
